UKWC Square Donation Form

1. Upload this folder to Netlify.
2. In Netlify, open Site configuration > Environment variables.
3. Add:
   SQUARE_ACCESS_TOKEN = your Square production access token
4. Redeploy the site.
5. Open the Netlify URL and test a small donation.
6. To show it inside Wix, add an Embed Website element and paste the Netlify URL.

Never place the Square access token inside index.html.
